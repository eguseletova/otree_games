A collection of theoretical or under-explored games. 

Games in this repo:

The E-Mail Game: this oTree app broadly replicates an experiment outlined in Colin Camerer's 2003 book "Behavioral Game Theory." Specifically, I replicate the email game described in section 5.3.5, which is a variation of a coordinated attack game proposed by Ariel Rubinstein. This is purely for practise purposes.
